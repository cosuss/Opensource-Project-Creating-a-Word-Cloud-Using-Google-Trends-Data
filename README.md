# Opensource-Project-Creating-a-Word-Cloud-Using-Google-Trends-Data

## 1. Project Overview
This project collects Google Trends-related data for a given keyword,
processes the text data, and visualizes the most frequent words using a word cloud.

## 3. Used Packages
- Python 3.11
- requests
- beautifulsoup4
- konlpy
- jpype1
- wordcloud
- matplotlib
